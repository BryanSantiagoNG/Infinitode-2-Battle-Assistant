Cómo usar el asistente

Guía completa para interpretar las recomendaciones.

⚙

1. Configura tu partida

Selecciona el modo, la fase, si hay boss y cuántos enemigos de cada tipo esperás. Cuanta más información des, más preciso será el análisis.

⚖

2. Ajusta las prioridades

Usa los sliders de "Prioridades" para indicar qué te importa más: DPS, bajo costo, control de masas, rendimiento en Endless, etc. Dos jugadores con estilos distintos recibirán recomendaciones distintas.

⚡

3. Analiza la partida

El motor de puntuación evalúa cada torre contra tus condiciones y asigna una puntuación de 0 a 100. Más alta = más recomendada para este escenario.

🏆

4. Interpreta el Top 10

Cada carta de torre muestra su puntuación y los motivos detrás de la recomendación. El color de la barra indica: Verde ≥ 75 · Amarillo ≥ 50 · Rojo < 50.

🔗

5. Aprovecha las sinergias

El asistente detecta combinaciones entre torres que amplifican el daño o el control (ej: Veneno + Congelador = más tiempo de efecto). Siempre hay una sección de sinergias.

⚠

6. Lee las advertencias

Si configuraste "Sin voladores" pero una de las torres top es Antiaérea, recibirás una advertencia automática. No las ignores.

📊

7. Usa el comparador

Si dudas entre dos torres, ve a la sección "Comparar" y ponlas cara a cara. Verás cada estadística en una barra normalizada para comparar fácilmente.

💾

8. Guarda y exporta

Guarda torres favoritas con ★ (persisten entre sesiones). Exporta el análisis a JSON para compartirlo o a PDF para imprimirlo.
